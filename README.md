# todew
An easy to use and pretty minimal command line utility for managing tasks, written entirely in python.

#   Installation

1. clone the repo:
```sh
git clone https://github.com/mr4dd/todew.git
```
-------------
2. cd into the directory:
```sh 
cd todew/src/
```
-------------
3. run 
```sh 
sh setup.sh
```
-------------
